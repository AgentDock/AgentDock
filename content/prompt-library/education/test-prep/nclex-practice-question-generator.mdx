# NCLEX Practice Question Generator

A new grad opens an old NCLEX question bank and finds row after row of plain four-option multiple choice, which is a problem, because the Next Generation NCLEX built its newer item types specifically to stop testing recall in isolation and start testing clinical judgment across a scenario. This prompt writes original NGN-style questions, never copied from any official NCSBN exam content.

Set [ITEM_TYPE:select:Extended multiple response,Matrix or grid,Cloze (drop-down),Hot spot,Bowtie,Trend,Mixed across all NGN item types] to choose the format, [CLINICAL_TOPIC] for the patient scenario or body system to build around, and [QUESTION_COUNT:number:5-20] for how many items to generate.

Build each item this way. One, write a brief but complete clinical scenario on [CLINICAL_TOPIC], vitals, relevant history, and presenting findings, enough detail to reason from. Two, construct the item in the exact structure of [ITEM_TYPE]: extended multiple response needs more than one correct answer among many options, matrix or grid needs a table of options crossed against categories, cloze needs sentence blanks filled from drop-down style option sets, hot spot needs a described location or finding to identify, bowtie needs a two-sided structure connecting likely conditions to actions and rationales through a central choice, and trend needs a data pattern across two or more time points. Three, apply polytomous scoring logic in the rationale, meaning explain not just the fully correct response but which partial answers would earn partial credit and why.

Close every set by naming which of the four NCLEX Client Needs categories, safe and effective care environment, health promotion and maintenance, psychosocial integrity, or physiological integrity, the scenario in [CLINICAL_TOPIC] falls under, since NGN items are still built against that same underlying framework even in their newer formats.
