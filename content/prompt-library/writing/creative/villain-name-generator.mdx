# Villain Name Generator

I'm naming a villain and want the name to carry actual menace instead of sounding like a placeholder bad guy.

This villain is a [VILLAIN_TYPE:select:mastermind schemer,brute force enforcer,corrupt authority figure,cult leader or fanatic,tragic or sympathetic villain,supernatural or monstrous].

The story is set in a [GENRE:select:fantasy,science fiction,contemporary thriller,superhero,horror,historical] world.

I want the name delivered as a [NAME_FORMAT:select:full name with title,single epithet or alias,codename or moniker,full name only].

Generate [COUNT:number:5-20] villain names that match the type, genre, and format I picked. Don't echo famous villains, nothing that sounds like a knockoff of the names everyone already knows, and skip on-the-nose evil vocabulary like Dark, Blood, or Doom unless the format is an epithet and the genre can carry it. Every name should be pronounceable on the first read, and names within one batch should sound like they come from the same world, a fantasy set drawn from one coherent language feel, a thriller set that reads like real names with the wrong kind of weight. For each name, add a short note on the specific threat or menace it conveys, whether that's calculated intelligence, brute force, corruption, fanaticism, tragedy, or something inhuman, and keep the tone consistent with the genre. A tragic or sympathetic villain should sound less cartoonish than a supernatural monster unless I asked for something else. Close with your top 3 picks and what makes each one land.
