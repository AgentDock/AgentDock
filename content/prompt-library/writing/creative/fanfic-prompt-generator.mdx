# Fanfic Prompt Generator

I want a batch of fanfic prompts for a specific fandom instead of generic original fiction ideas.

The fandom is [FANDOM] (name the show, book, game, or franchise).

My pairing or focus is [PAIRING_OR_FOCUS?] (a specific pairing, a friendship, or a single character you want centered)

The trope I'm building around is [TROPE:select:enemies to lovers,found family,alternate universe,fix-it or canon divergence,hurt and comfort,secret relationship,time travel or body swap,no specific trope].

The tone should land as [TONE:select:fluffy and lighthearted,angsty and emotional,slow burn,fast and plot-driven].

Generate [COUNT:number:3-10] fic prompt ideas grounded in [FANDOM] and its actual characters, not generic placeholders. For each prompt, give a one-line premise and name the trope or tone it leans into. If a prompt needs a specific canon-divergence point or an alternate-universe setup to work, say so directly. Vary where each prompt sits in the story, some right after canon events, some in a stretch of time canon never covers, so the batch doesn't repeat the same jumping-off point.
