# Fantasy Name Generator

I need a fantasy name and I'm not sure yet which specific tool fits, so I want one generator that covers any category: a person, a place, an object, a creature, or a title.

The category is [NAME_CATEGORY:select:a person or character,a place - city,town,or kingdom,an object or artifact,a creature or race,a title or organization].

The tone I want is [TONE:select:noble and elegant,dark and ominous,whimsical and light,ancient and mythic,rugged and wild].

Any detail worth building the name around: [CONTEXT?] (a role, a region, a specific story beat)

Generate [COUNT:number:5-20] fantasy names matching my chosen category and tone. For each one, add a short note on what it evokes and why it fits the category, a person's name should read differently from a kingdom's name even at the same tone setting. Vary the length and structure across the batch so nothing feels like a copy of the last entry. If I need names for a specific fantasy race like elves or dwarves, or a narrower category like a tavern or a spell, say so at the end and suggest a specialized generator instead for more depth. Close with your top 3 picks and why they stand out.
