# NCLEX Select All That Apply Practice Generator

Ask nursing students which NCLEX question format they dread most and select all that apply wins by a wide margin, because there's no way to guess your way to a plausible-sounding answer when the number of correct options isn't given. Under the Next Generation NCLEX this format lives on as extended multiple response, still asking you to select every option that fits, now scored with partial credit instead of all-or-nothing. This prompt writes original select all that apply items, never copied from any official exam bank.

Set [CLINICAL_TOPIC] for the patient scenario or nursing concept to build around and [QUESTION_COUNT:number:5-20] for how many items to generate.

Work through this method for each item. First, write a clinical scenario or a nursing action prompt on [CLINICAL_TOPIC] with five to eight answer options, where more than one but not all of them are correct, since a set that's always exactly two or exactly four correct teaches pattern guessing instead of clinical reasoning. Second, mark which options are correct and which are distractors, and for every distractor, explain specifically what makes it wrong, not related enough, contraindicated, or correct information applied to the wrong situation. Third, for every correct option, explain what makes it right in one sentence tied to the specific scenario, not a general fact about the topic.

Close each item with a note on how extended multiple response partial-credit scoring would treat a response that got most but not all of the correct options right, so the practice builds accurate intuition for the actual scoring, not the all-or-nothing instinct older SATA formats trained.
