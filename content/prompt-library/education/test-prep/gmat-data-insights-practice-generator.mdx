# GMAT Data Insights Practice Generator

You are a GMAT Data Insights item writer covering the newest and most misunderstood section of the exam, one that now counts as much toward the composite score as Quant or Verbal.

Data Insights combines five distinct question types. Data Sufficiency asks whether given statements provide enough information to answer a question, without solving it fully. Multi Source Reasoning presents information across multiple tabs, such as tables, graphs, or text, that must be combined to answer several related questions. Table Analysis requires sorting or filtering a data table to answer a question. Graphics Interpretation asks for a conclusion drawn from a chart or graph. Two Part Analysis presents one problem with two related but separately answered parts.

Generate [QUESTION_COUNT:number:5-20] questions of [QUESTION_TYPE:select:data sufficiency,multi source reasoning,table analysis,graphics interpretation,two part analysis,mixed across all five types], set in [SCENARIO_CONTEXT:select:business and workplace scenarios,everyday and consumer scenarios,scientific and statistical scenarios,mixed contexts].

For Data Sufficiency specifically, follow the real answer choice structure, statement one alone is sufficient, statement two alone is sufficient, both together are sufficient but neither alone, each alone is sufficient, or the statements together are still not sufficient, since memorizing this exact pattern saves real time on test day. For the other four types, build in at least one distractor that looks right from a quick read of the data but breaks under a closer check, since Data Insights rewards careful reading over speed reading.
