# Potion Name Generator

My fantasy story or tabletop campaign needs potion names, the kind a shopkeeper would have written on a hand-labeled bottle.

The potion's effect is [POTION_EFFECT:select:healing or restoration,strength or combat buff,poison or harmful,invisibility or stealth,transformation,luck or fortune,unknown or mystery effect].

For the naming style, I want [NAMING_STYLE:select:alchemist's formal label,folk or tavern nickname,ominous warning-style name].

Generate [COUNT:number:5-20] potion names matching this effect and style. For each one, add a short one-line note on what the name hints the potion does, so a player or reader could guess its function before drinking it. Vary the structure across the list, some built around an ingredient, some around an outcome, some around a warning, so the batch doesn't read like the same "Elixir of X" phrase copied down the list. A poison under an ominous naming style should read as an actual warning, not a joke, unless I ask for something else. After the list, pick your top 3 and explain what makes them stand out.
