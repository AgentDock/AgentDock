# Command and Imperative Practice Generator

The imperative is the one verb form built almost entirely out of exceptions. French drops the final -s from -er verb commands, mange, not manges, except it puts that same -s right back when the pronoun en or y follows, manges-en. Italian's negative tu command doesn't conjugate at all, it borrows the plain infinitive, non parlare, because the regular conjugated form would collapse into a statement instead of a command. A learner who assumes the imperative just follows the regular present tense pattern gets it wrong constantly, since almost none of these languages let it.
