# TEAS Science Practice Generator

Science is the section most TEAS 7 takers name as the hardest, and it's usually not because the material is exotic, it's because anatomy, physiology, and scientific reasoning cover more ground than the other three sections combined and there's less time to relearn all of it from scratch. This prompt writes original practice questions across that ground, never copied from ATI's official test bank.

Set [SCIENCE_TOPIC:select:Human Anatomy and Physiology,Life and Physical Sciences,Scientific Reasoning,Mixed review across all topics] to choose what to drill and [QUESTION_COUNT:number:10-40] for how many questions to generate.

Start with a short diagnostic framing before writing any questions. If [SCIENCE_TOPIC] is mixed review, open with one question from each major area, body systems, cellular biology, basic chemistry, and scientific method or experimental design, so a first pass surfaces which area needs the most repetition before the rest of the set gets built. If [SCIENCE_TOPIC] is a single area, skip the diagnostic and go straight into full-depth questions on that topic.

Write [QUESTION_COUNT] questions total, mixing recall items, name this structure, identify this process, with applied reasoning items that describe a scenario, a symptom pattern or an experimental result, and ask what it indicates. Provide a full answer key with an explanation for each item that states the underlying concept in one or two sentences, since on this section the concept usually matters more than the specific fact being tested.

End with a short list of the two or three topics inside [SCIENCE_TOPIC] that generated the most questions, so a student can see at a glance where this particular set concentrated and decide whether to run it again with a narrower focus.
