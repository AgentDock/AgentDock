# CompTIA A+ Practice Test Generator

Which of the two required A+ exams needs more of your study time right now, Core 1's hardware, mobile devices, networking, and cloud fundamentals, or Core 2's operating systems, security, and software troubleshooting? Both have to be passed to earn the full certification, and most career-changers walking into IT for the first time find one noticeably harder than the other. This prompt writes original A+ style practice questions, never copied from CompTIA's official exam content.

Set [EXAM_FOCUS:select:Core 1 (220-1201) Mobile devices hardware networking and cloud,Core 2 (220-1202) Operating systems security and troubleshooting,Mixed across both exams] and [QUESTION_COUNT:number:10-40] for how many questions to generate.

Work through this method. One, if [EXAM_FOCUS] is Core 1, write questions on mobile device configuration, PC hardware components, networking fundamentals, and basic virtualization and cloud concepts. If [EXAM_FOCUS] is Core 2, write questions on operating system installation and configuration, security fundamentals and threat mitigation, software troubleshooting, and operational procedures like documentation and safety practices. If mixed, split [QUESTION_COUNT] evenly between both exams' domains. Two, vary the question format across the set, standard single-answer multiple choice, multiple-response questions with more than one correct answer, and at least one scenario-based item per set that describes a troubleshooting situation and asks for the most likely next diagnostic step, mirroring the exam's performance-based question style in text form. Three, provide a full answer key with a one-sentence rationale for each item explaining the underlying concept, not just the correct choice.

Close with a short note on which exam, Core 1 or Core 2, this particular career path typically leans on more heavily day to day, hardware-focused field roles versus help desk and security-adjacent roles, so study time can be weighted toward the job being targeted, not just the exam sequence.
