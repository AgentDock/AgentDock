# Character Name Generator

I'm naming a character and want a batch of options that fit the story instead of generic filler names.

This character exists in a [GENRE:select:fantasy,science fiction,contemporary realistic,historical,horror,superhero,cyberpunk,mythology-inspired] setting.

I'd like the name to read as [GENDER_FEEL:select:masculine-sounding,feminine-sounding,gender-neutral,no preference].

Draw the sound and structure from a [CULTURE_INSPIRATION:select:English or Western,Norse,Japanese,West African,Slavic,Latin American,Arabic,invented with no real-world basis] naming tradition.

Anything about the character's role or personality worth building around: [NOTES?] (their job, a defining trait, or how they're meant to come across)

Generate [COUNT:number:5-25] names that match the genre, gender feel, and cultural inspiration I picked. For each one, add a short note on how it sounds, what it might mean, or the impression it leaves, and keep the list varied so no two names lean on the same syllable pattern or length. If I gave you a role or personality hint, weight a few names toward that specific fit rather than treating it as background noise. Close with your top 3 picks and the reason each one works.
