# TEAS Reading Practice Generator

A test taker sits down for the TEAS 7 Reading section already knowing the general skills it covers and still gets caught off guard, because "practice reading comprehension" without knowing which specific skill a missed question tested doesn't actually tell you what to fix. This tool writes an original passage and a tagged question set so every miss points back to a named skill instead of a vague sense of "reading is hard."

Set [PASSAGE_TOPIC] to a subject area for the passage, health and science topics work well since the real section leans that way, [PASSAGE_LENGTH:select:Short (150-250 words),Medium (250-400 words),Long (400-600 words)] for how long it runs, and [QUESTION_COUNT:number:5-15] for how many questions to build from it.

Score every question you write against this rubric before including it: does it test key ideas and details, meaning locating the main idea, a supporting detail, or a stated fact in the passage, does it test craft and structure, meaning word choice, author's purpose, or how the passage is organized, or does it test integration of knowledge and ideas, meaning drawing an inference, evaluating an argument, or reading information presented in a chart or graphic tied to the passage. Tag each question with which category it falls under directly above its answer explanation.

Write the passage first, on [PASSAGE_TOPIC], at the [PASSAGE_LENGTH] you selected, using original content, not an excerpt from any published source. Then generate [QUESTION_COUNT] questions against it, distributed across all three rubric categories rather than clustering in whichever type is easiest to write. Close the answer key with a one-line tally of how many questions landed in each category, so a skipped category becomes visible at a glance.
