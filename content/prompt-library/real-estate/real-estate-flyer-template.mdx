# Real Estate Flyer Template

You are a real estate marketing copywriter. I need you to create complete flyer content for a property listing that is ready to drop into any design tool or printed layout. The property is a [PROPERTY_TYPE:select:single-family home,townhouse,condominium,duplex,multi-family,luxury estate,commercial property] located at [PROPERTY_ADDRESS] in [CITY_STATE]. The listing price is [LISTING_PRICE]. The property has [BEDROOMS:number:1-10] bedrooms, [BATHROOMS:number:1-10] bathrooms, and [SQUARE_FEET:number:400-20000] square feet of living space.

The top selling features of this property are [KEY_FEATURES]. The target buyer profile is [TARGET_BUYER:select:first-time buyers,move-up families,downsizers and retirees,luxury buyers,investors,young professionals]. The flyer format should be [FLYER_FORMAT:select:single-page listing flyer,open house announcement,just listed postcard,just sold postcard,price reduction notice,coming soon teaser]. My name and contact details are [AGENT_INFO].

Generate the following flyer sections. Start with a headline that grabs attention and creates emotional appeal for the target buyer. Write a property description of 60 to 80 words that highlights the lifestyle this home offers, not just square footage and room counts. Create a bulleted feature list of 6 to 8 items covering the most marketable aspects of the property, lot, and location. Add a neighborhood snapshot of 2 to 3 sentences covering schools, commute times, dining, or parks that matter to the target buyer profile.

Include a clear call to action with next steps for interested buyers, such as scheduling a showing, attending an open house, or scanning a QR code for a virtual tour. End with the agent contact block formatted for print.

Keep all copy concise and scannable. Use active language that helps buyers picture themselves in the home. Avoid generic phrases like "must see" or "won't last long" and replace them with specific, benefit-driven statements tied to the actual features of this property.
